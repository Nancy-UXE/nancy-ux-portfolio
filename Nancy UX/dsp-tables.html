Ads Tables


Tables are a great way to display quantitative information, among other usages. Whenever we have more than one set of values that have a direct relationship, we may use a table to organize the data.

For example in Ads Platform tables are used extensively, as we were planning to use the table to look up one or more particular values. Also the information will be used to examine a set of quantitative values as a whole to spot patterns.

I worked on many tables features, below are some of them:

  • Table Filter
  • Inline Editable Cell
  • Pacing Visualization
  • Right Rail
Table Filter
Description of Feature Recap

This feature is an important step towards improving the overall ease of use of the Ads Platform. Currently, on the line grid, Ads Platform users are only able to filter lines using a search box that queries line names or line IDs. This functionality works for basic searches, but doesn’t allow users to perform a more intelligent analysis based on the other data present in the grid. With the Advanced Filters/Search for the Lines Grid release, all Ads Platform users can now apply several filters to the Lines grid, greatly increasing the usability of the grid and all its glorious data!

What is it? What are the key benefits?

Ads Platform users (and frankly, the internet) are accustomed to drilling down through lots of information to find what they need and have become very efficient at doing so. They want to find things quickly and with the minimum number of clicks. By helping users drill down to their needs with ease, eliminates the need for numerous search queries just to find something specific and also enhances the overall user experience within our platform. Inline Editable Cell
Description of Feature Recap

The introduction of In-Line editing is to improve the overall usability of Ads Platform+ grid views and introduce interactivity within them. This includes workflow improvements for list/grid views on Ads Platform+, focusing more towards - Advertiser, Campaign, and Line list/grid views. This concludes the objective to have all main grid levels of Ads Platform supported in some format with in-line editing capabilities.

Overall Improvements At All Ads Platform Levels Include:
  • New grid presentation
  • New metric columns added to make the list views with more detailed information
  • Connect different grids in one place.
  • Easy validation, with guidance about the corret values
  • Cleaner way to depict long names
  • Ability for users to edit the metadata list view fields inline and an explicit save operation for inline edits
  • Editable fields are highlighted/represented in a different way in list view vs the non editable fields for easy identification

Pacing Visualization
Description of Feature Recap

With this release, we bring to the users an easy way to track how their campaigns/lines budget is being spent, how their campaigns have been pacing historically and the current pacing values with data visualization thus helping users to easily monitor overall campaign health. Users will also be able to monitor spend/pacing of schedules in a comprehensive view and in fact edit multiple schedules for an object from the grid itself instead of having to go to the object property page.

What is it? What are the key benefits?

Users who handle daily campaign operations need a user-friendly management interface to keep track of their campaign delivery . One of the most common tasks is ensuring that campaigns are properly pacing based on the flight dates and pre-defined budgets. Anything a platform can do to assist in making that task easier through helpful visualizations, goes a long way in making the user’s job easier and their experience on our platform delightful.

User Feedback:
  • An example of how these new features support the Ads Platform revenue growth is the Pacing Visualization feature that was Beta Released for the Ads Platform on September 14.
    This feature provides a user-friendly management interface to keep track of campaign delivery, (pacing and spend).
  • We have received some great feedback on this feature from a client, Cadreon, "I love it, everything is more visible... everything is in one place. Makes it so much easier for me". We got very similar feedback from most of our clients.
  • Seeing as the current Amazon campaign has spent about $160k in 10 days, the Platform solutions team see Pacing Visualization as a great success and it will have definitely helped improve the stickiness of the Ads Platform.

Right Rail
Description of Feature Recap

This feature is one of the most important features that we have added to the tables in Ads Platform. It is a quick view of information, graphs and other data that is important for users to have quick acess to. Very nice about this feature, that it saves a lot of clicks and waiting, in order to view most needed information in one place by one click.

Feature Wins
  • Made the access to the information and data representation really quick
  • Saved a lot of time, and cut the user journey in order to reach the information in less than half.
  • All in one place, where we could add more information, without the need to change the page
  • Helped in faster and quicker decisions
