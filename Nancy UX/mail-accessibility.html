Mail Accessibility


Many aspects of accessibility are fairly easy to understand and implement. Some accessibility solutions Yahoo Mail app are more complex and take more knowledge to implement.

My contribution includes:

  • Worked with the Accessibility Lab closely
  • Tested Accessibility using differ tools
  • Led the Yahoo Mail Accessibility project successfully
  • Applied Keyboard shortcut Accessibility keys
  • Applied HTML Accessibility tags (Forms, Table, Tabs)
  • Worked on Accessibility for different themes
  • Structured the CSS in order to have tokens, and veriables, so the theme swith is easier

Making Yahoo Mail as a more inclusive place for everyone, a good accessibility strategy also has business benefits. Accessibility is a component of design and development that touches on almost every element of the Yahoo Mail creation. Accessible Yahoo Mail can have better usability, reduced maintenance costs, increased audience reach.

Below are some of the itmes we took in consideration while apply Yahoo Mail accessibility:

  • Structure and semantics (Page title, headings, tables, etc)
  • Links and navigation
  • Images and visual design
  • User input, forms, and dynamic content
  • Keyboard shortcuts and Tab sequance.
  • Appearance in different browsers, text sizes, screen sizes
  • Color contrast and compatibility with different themes

Dark Themes

As Yahoo mail deals with huge number of users, giving them the option to choose the theme and the dark theme, was one of the essentials while working on the user experience for Yahoo Mail.
However, Dark mode users find reading easier in low light with less eye strain. They also claim it helps them fall asleep quickly and stay asleep longer. Users claim it reduces screen headaches as well.




Dark themes required different color palettes, in order to ensure the colors contrast compatibility. We developed a switch color rules between light and dark theme, below is the basic one, as we had to develop different ones, to cover the other color themes. Following the same system, made it easier for both engineers and designers to apply accessible themes.
